Jogo da corrida

Este projeto é um jogo simples e divertido em que animais como cachorro, tartaruga, galinha e girafa competem para ver quem cruza a linha de chegada primeiro. Cada vez que o jogo é iniciado, os movimentos dos personagens são aleatórios, o que torna cada corrida diferente. Esse projeto me mostrou que é possível aprender se divertindo e usar a programação como ferramenta para criar jogos interativos e criativos.

Projeto Monalisa

A Monalisa Moderna é uma releitura criativa feita com formas geométricas simples. Usando círculos, triângulos e linhas, montei um rosto com expressão e estilo próprio, mostrando como é possível fazer arte com código. Esse projeto me ensinou que programação não é só números e comandos, mas também uma forma de expressar ideias, explorar estilos visuais e transformar a tela em uma tela de pintura digital.
